## The Technology Behind It

For the tech enthusiasts:
- **AI Model:** Google Gemini 2.0 Flash (latest model with multimodal capabilities)
- **Frontend:** React.js with Material-UI for modern, responsive design
- **PDF Processing:** PDF.js for client-side rendering and text extraction
- **Caching:** IndexedDB for intelligent local storage of AI responses
- **Storage:** Firebase Firestore (cloud sync) + IndexedDB (offline-first)
- **Authentication:** Google Sign-In for seamless sync
- **Text-to-Speech:** Web Speech API with multi-language support
- **Smart Chunking:** Automatic text splitting to handle token limits
- **Context-Aware AI:** Passes summaries of prior pages for exercise solutions
- **Bilingual Architecture:** All prompts designed for dual-language output
- **Privacy-First:** Processing happens via your own API key

**Open Source:** The entire project is freely available on GitHub. No paywalls, no subscriptions. Clone it, customize it, deploy it!

## Lessons from That Sunday Morning

1. **Listen to Your Users:** My daughter knew exactly what she needed. I just helped build it.

2. **Start with Real Problems:** Not "what's a cool AI project?" but "how can AI solve this specific struggle?"

3. **Make It Accessible:** Free tools, simple setup, no coding required. If a 7th grader can use it, anyone can.

4. **Design for Bilinguals:** Many students read in one language but think in another. Build for that reality.

5. **Empower, Don't Replace:** The goal wasn't to replace her learning process ‚Äì it was to enhance it.
