## üöÄ Getting Started

### **Step 1: Upload Your PDF**

1. Click **"Open My Library"** on the Dashboard
2. Click **"+ Add PDF"** button
3. Choose a PDF file from your computer
4. Wait for the upload to complete
5. Click on the PDF to start reading

### **Step 2: Configure API Keys**

Ekamanam uses AI providers like Google Gemini, Groq, and Perplexity. You need to add API keys to enable AI features.

**To add API keys:**

1. Click the **Settings** icon (‚öôÔ∏è) in the top-right
2. Go to **"API Keys"** tab
3. Add your API keys:
   - **Google Gemini** (Recommended for regional languages)
   - **Groq** (Fast responses)
   - **Perplexity** (Internet-connected answers)
4. Click **"Save"**

**Getting API Keys:**

- **Google Gemini**: https://aistudio.google.com/app/apikey (Free tier available)
- **Groq**: https://console.groq.com/keys (Free tier available)
- **Perplexity**: https://www.perplexity.ai/settings/api (Paid)

### **Step 3: Set Up Your Profile**

1. Open **Settings** ‚Üí **"General"** tab
2. Enter:
   - **Student Name**
   - **Parent Name**
   - **Parent Email** (for OTP access to Admin Dashboard)
3. Click **"Save Profile"**

### **Step 4: Choose Your Preferences**

**Voice Settings:**
- Settings ‚Üí **"Voice & Speech"** tab
- Select your preferred voice (Indian English, Hindi, Telugu, Tamil)
- Test each voice before selecting

**Theme:**
- Click the **theme toggle** (üåô/‚òÄÔ∏è) in the header
- Switch between Light and Dark mode

**Language:**
- The app auto-detects your PDF language
- AI responses adapt to your content's language

## üìñ How to Use Each Feature

### **1. Teacher Mode** üéì

**What it does:** Provides comprehensive explanations of entire pages or chapters.

**How to use:**
1. Open a PDF in your library
2. Navigate to the page you want to learn
3. Click **"Teacher Mode"** tab
4. Select scope:
   - **"This Page"** - Explain current page only
   - **"Entire Chapter"** - Explain multiple pages
5. Click **"Generate Content"**

**What you get:**
- Summary of the topic
- Key points (bullet list)
- Detailed explanation
- Real-world examples
- Exam tips

**Pro Tips:**
- Use "This Page" for quick understanding
- Use "Entire Chapter" for comprehensive study
- Click **"Listen"** to hear the explanation
- Click **"Add to Notes"** to save content
- Use **"English"** button to translate regional language content

---

### **2. Smart Explain** üí°

**What it does:** Deep dives into specific concepts with exercises.

**How to use:**
1. **Select text** in the PDF (drag your mouse over the text)
2. Click **"Smart Explain"** tab
3. Choose scope:
   - **"Selection Only"** - Explain only the selected text
   - **"Selection + Page"** - Include page context
4. Click **"Analyze & Explain"**

**What you get:**
- Detailed explanation
- Analogy/real-world connection
- Practice exercises with answers
- Step-by-step solutions
- Important notes
- Hints for better understanding

**Pro Tips:**
- Select key terms or concepts you don't understand
- Try exercises before revealing answers
- Listen to bilingual answers (original + English)
- Expand exercises to see step-by-step solutions

---

### **3. Activities** üéØ

**What it does:** Generates practice questions from your study material.

**How to use:**
1. Click **"Activities"** tab
2. Select scope (page or chapter)
3. Click **"Generate Activities"**

**What you get:**
- Fill in the blanks
- Short answer questions
- True/False questions
- Definition questions

**Pro Tips:**
- Practice before exams
- Try answering before checking solutions
- Use "Add to Notes" to save questions

---

### **4. Multilingual (Word-by-Word Analysis)** üåç

**What it does:** Analyzes text in regional languages word by word.

**How to use:**
1. Open a PDF in Hindi, Telugu, Tamil, etc.
2. Click **"Multilingual"** tab
3. The page is automatically analyzed
4. Click **"Start Analysis"** if needed

**What you get (for each word):**
- Original word with pronunciation
- Part of speech
- English meaning
- Pronunciation guide
- Example usage
- **Listen button** to hear pronunciation

**Pro Tips:**
- Great for learning new vocabulary
- Click "Listen" to improve pronunciation
- Works best with Hindi, Telugu, Tamil text
- Automatically detects language

---

### **5. Exam Prep** üìù

**What it does:** Generates exam-style questions from your content.

**How to use:**
1. Click **"Exam Prep"** tab
2. Select scope (page or chapter)
3. Click **"Generate Exam Prep"**

**What you get:**
- **MCQs** (Multiple choice questions)
  - 4 options per question
  - Correct answer highlighted in green
  - Explanation for each answer
- **Short Answer Questions** (2-3 sentence answers)
- **Long Answer Questions**
  - Click "Generate Answer" for model answer
  - Detailed, exam-ready responses

**Pro Tips:**
- Practice MCQs first
- Try writing short answers before revealing
- Use long answers as templates for your own
- Add questions to Notes for revision

## üÜò Troubleshooting

### **Problem: AI features not working**
**Solution:**
- Check if API keys are configured (Settings ‚Üí API Keys)
- Verify API keys are valid (test on provider website)
- Check internet connection

### **Problem: PDF not loading**
**Solution:**
- Check file size (very large PDFs may take time)
- Ensure PDF is not password-protected
- Try a different browser (Chrome recommended)

### **Problem: Regional language not detected**
**Solution:**
- App auto-detects language
- If wrong, Teacher Mode will still work
- Multilingual tab works best with Unicode fonts

### **Problem: Voice not working**
**Solution:**
- Check browser supports text-to-speech
- Allow microphone/audio permissions
- Try different voice in Settings

### **Problem: Dark mode looks broken**
**Solution:**
- This should be fixed in v3.5.2
- Try refreshing the page (Ctrl+R or Cmd+R)
- Report issue if it persists

### **Problem: Vyonn not responding**
**Solution:**
- Check Gemini/Perplexity API keys
- Verify internet connection
- Clear browser cache
